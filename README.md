Construction Company Website

A modern and responsive construction company website built with React, Vite, and TailwindCSS. The project is designed to showcase company services, completed projects, company information, and contact options through a clean and professional user interface.

Features
Responsive design for mobile, tablet, and desktop devices
Modern construction-company style UI
Home, About, Services, Projects, Contact and Privacy Policy pages
Featured projects and project filtering by category
Reusable components and scalable project architecture
Professional contact section with company information
Project details structure prepared for future expansion
Clean and maintainable component structure
Technologies Used
React
Vite
TailwindCSS
React Router
Font Awesome
Future Improvements
Project details pages
Multi-language support (MK / EN)
Contact form email functionality
Admin dashboard and backend integration
Dynamic project management system
Image galleries and before/after project previews
Goal

The goal of this project is to build a professional business website that can later evolve into a full project management platform with backend functionality and an admin system.
